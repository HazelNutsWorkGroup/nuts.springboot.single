# <a id="info">1. 介绍</a>
「Nuts-SpringBoot-Single」基于org.springframework.boot的2.2.5.RELEASE版本进行开发的，并采用Gradle进行项目构建。
由于Gradle是初次使用，且org.springframework.boot采用的版本也是项目开发时最新的版本，不推荐应用到实际生产项目中。
过程中参考的开源项目会列到参考文献章节中，感谢其作者的无私奉献。
